PRIVACY POLICY — QZERO

Last Updated: 24/11/2025

This document explains how QZero ("we", "the application", "the platform") collects, uses, and protects personal data in compliance with GDPR and applicable Portuguese law.

Controller: Independent developer (individual operator).

1. Data We Collect

For All Users:

  • Email address (for authentication)
  • Password (encrypted)
  • Account type (Customer or Business)
  • Geolocation (only when user allows, used for nearby business search)

For Business Accounts:

  • Business name, address, description
  • Operating hours, services offered
  • Staff accounts (if team features are used)
  • Payment method (through Stripe)

Usage Data:

  • Queue and booking activity
  • Reviews left by customers
  • Basic analytics (anonymous and aggregated)

2. Legal Basis for Processing

We process data based on:

  • Contract: Necessary to provide services
  • Consent: For optional features (e.g., geolocation)
  • Legal obligations: Financial/payment records
  • Legitimate interest: Service improvement, security

3. Purpose of Data Usage

  • Provide the core service (queue management, bookings)
  • Authenticate users
  • Process payments (via Stripe)
  • Communicate essential notifications
  • Improve the platform through analytics

4. Data Sharing

We share data only with:

  • Stripe (payment provider, PCI-DSS compliant)
  • Hosting providers (to run the platform)
  • Legal authorities (if required by law)

We do NOT sell personal data to third parties.

5. Cookies & Local Storage

We use:

  • Cookies for authentication (session tokens)
  • Local Storage for user preferences and language settings

We do NOT use advertising or tracking cookies.

6. Security Measures

  • Password encryption
  • HTTPS on all communications
  • Secure authentication tokens
  • Access-restricted backend infrastructure
  • Internal auditing

7. User Rights

Users may, at any time:

  • Request access to their data
  • Request correction
  • Request deletion (right to be forgotten)
  • Request data export
  • Withdraw consent
  • File complaints with the Portuguese Data Protection Authority (CNPD)

Requests should be made to: suporteqzero@gmail.com

8. Data Retention

  • Account data → retained until account deletion
  • Queue/booking history → retained while account is active
  • Technical logs → up to 90 days
  • Payment records → according to Stripe's legal requirements

9. International Transfers

If data is processed outside the EU, it is only through GDPR-compliant providers (such as Stripe or hosting providers with Standard Contractual Clauses).

10. Changes to this Policy

The latest version will always be available before registration or account creation.
