MULTI-IDIOMA (IMPLEMENTAÇÃO INTERNA)
──────────────────────────────
Cria um sistema de internacionalização usando i18n baseado no stack existente (React, Vue ou outro — detecta automaticamente). Ele deve:

- Carregar ficheiros JSON estruturados assim:

/locales
   /pt
      translation.json
   /en
      translation.json
   /es
      translation.json
   /fr
      translation.json

- Cada JSON deve conter todas as strings dinâmicas da UI.
- Nada de texto hardcoded nos componentes → substituir por chaves de tradução.
- Detetar idioma padrão via `navigator.language`.
- Guardar idioma escolhido no perfil do utilizador ou localStorage.
- Adicionar função global `changeLanguage()`.

NÃO alterar textos de API, apenas UI.
